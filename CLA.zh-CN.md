# Cebian 个人贡献者许可协议

> 方便阅读。若中英文之间发生任何歧义或冲突，**以 [CLA.md](CLA.md)

感谢你有意为 Cebian（"**本项目**"）贡献代码。本项目由
<https://github.com/maotoumao/Cebian> 所在 GitHub 仓库的拥有者维护
（当前由名为 "maotoumao" 的 GitHub 用户运营），及其继任者和受让人
（统称 "**维护者**"）。

我们知道，读一份法律文件并不是贡献开源项目里最有趣的环节。这份 CLA
我们尽量保持简短，并采用了广为人知的 Apache CLA 作为模板，希望你
能快速读完，把时间留给真正有趣的部分——写代码。如有任何不清楚的
地方，欢迎开 issue 与我们讨论。

本协议（"**本协议**"）记录贡献者授予维护者的权利。要使本协议生效，
请按 [CONTRIBUTING.zh-CN.md](CONTRIBUTING.zh-CN.md) 的说明完成签署
流程。鉴于这是一份法律文件，请在同意前仔细阅读。

本协议基于 Apache 软件基金会个人贡献者许可协议 V2.0 制定，并针对
单一维护者的项目做了相应调整。

## 1. 定义

**"你"**（或 **"你的"**）指版权所有人，或版权所有人授权、与维护者
订立本协议的法律实体。对于法律实体，提交贡献的实体以及控制该实体、
受该实体控制或与该实体处于共同控制下的所有其他实体，都被视为同一
贡献者。

**"贡献"** 指任何原创作品，包括对已有作品的修改或补充，由你有意
提交给维护者、以便纳入本项目或其文档的内容。就本定义而言，
"提交" 指以电子、口头或书面形式发送给维护者或其代表的任何交流，
包括但不限于在由维护者或其代表管理的电子邮件列表、源代码控制系统
和问题跟踪系统上进行的交流，用于讨论和改进本项目；但不包括你以
书面形式明确标注或指定为 "非贡献" 的交流。

## 2. 版权许可授予

在遵守本协议条款和条件的前提下，你在此授予维护者及维护者分发软件的
接收者一项永久的、全球范围的、非独占的、免费的、免版税的、不可撤销
的版权许可，以复制、制作衍生作品、公开展示、公开表演、再许可和分发
你的贡献以及该等衍生作品。

你进一步授予维护者**再许可**你的贡献的权利，维护者可将你的贡献按其
选择的任何许可证发布，包括专有或商业许可证，可作为本项目当时的开源
许可证的补充或替代。这包括更改本项目（或作为本项目一部分的你的
贡献）所依据的开源许可证的权利，以及同时以多个许可证分发本项目的
权利。你理解并同意，维护者可在对你没有任何义务的前提下，以一个或
多个许可证提供本项目——或其任何部分，包括你的贡献——
（例如，以本项目的开源许可证以及面向付费客户的商业许可证同时提供）。

## 3. 专利许可授予

在遵守本协议条款和条件的前提下，你在此授予维护者及维护者分发软件的
接收者一项永久的、全球范围的、非独占的、免费的、免版税的、不可撤销
（除本节另有规定外）的专利许可，以制造、委托制造、使用、许诺销售、
销售、进口和以其他方式转让本项目；该许可仅适用于可由你许可、且被
你的贡献本身或你的贡献与被提交到的本项目相结合所必然侵犯的专利权
利要求。如任何实体对你或任何其他实体提起专利诉讼（包括在诉讼中
提起反诉或交叉诉讼），指控你的贡献或你所贡献的本项目构成直接或
共同专利侵权，则本协议项下为该实体授予的、就该贡献或本项目的任何
专利许可，将自该等诉讼提起之日起终止。

## 4. 陈述

你陈述：

- 你有合法权利授予上述许可。如果你的雇主对你创建的包含你贡献的
  知识产权享有权利，你陈述你已获得以雇主名义作出贡献的许可、你的
  雇主已放弃对你贡献的该等权利，或你的雇主已与维护者签署了单独的
  协议。
- 你的每一项贡献均为你的原创作品（关于代他人提交的情形，见第 5 节）。
- 你的贡献提交包含你个人知悉的、与你贡献任何部分相关的任何第三方
  许可或其他限制（包括但不限于相关专利和商标）的完整细节。

## 5. 代他人提交

如你希望提交并非你原创的作品，你可以将其与任何贡献分开提交给维护者，
并标明其来源以及你个人知悉的任何许可或其他限制（包括但不限于相关
专利、商标和许可协议）的完整细节，并在作品上明显标注 "代第三方提交：
[此处注明姓名]"。

## 6. 支持

不要求你为你的贡献提供支持，除非你愿意提供。你可以免费提供支持、
收费提供支持，或不提供支持。

## 7. 免责声明

除非适用法律要求或书面约定，你按 "**现状**" 提供你的贡献，**不附有
任何明示或默示的担保或条件**，包括但不限于所有权、不侵权、适销性或
适用于特定用途的任何担保或条件。

## 8. 情况变更通知

如你知悉任何事实或情况将使上述陈述在任何方面变得不准确，你同意通知
维护者。
