You can install Cebian in any of the ways below. For everyday use, installing from an extension store is recommended; if you want the latest version right away, you can download it from GitHub Release.

## Install from an extension store

Chrome, Edge, and most Chromium-based browsers can install from their extension stores:

- [Chrome Web Store](https://chromewebstore.google.com/detail/dnfmklmdbipiahdjiogaoemjaiodomdf)
- [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/kakfgcmbiacmclkjboijakkhlhioopcd)

Open the corresponding page and click the install button. Once installed, the Cebian icon will appear in your browser toolbar; click it to open the sidebar.

> Store versions go through review, so they may update a little later than GitHub Release.

## Install from GitHub Release

Updates are also published to GitHub at the same time. You can download the latest version from Releases:

[Cebian GitHub Releases](https://github.com/maotoumao/Cebian/releases)

The installation steps are:

1. Download `cebian-{version}-chrome.zip`
2. Unzip it into a fixed directory
3. Open your browser's extension management page
4. Enable "Developer mode"
5. Click "Load unpacked"
6. Select the directory you just unzipped

> The Release also includes a Firefox package. If you want to install manually on Firefox, you can download the corresponding `firefox.zip`, but we won't go into the Firefox installation process here.

## Choosing an installation method

- Everyday use: the extension store is the easiest
    - Edge users: prefer the Edge Add-ons store
    - Chrome / Chromium users: prefer the Chrome Web Store

- Want the latest version: GitHub Release is usually fastest

## After installation

Once installed, click the Cebian icon in your browser toolbar to open the sidebar. The first time you use it, you need to configure an AI provider, and then you can start chatting.
