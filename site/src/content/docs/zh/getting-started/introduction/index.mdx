Cebian 是一个常驻浏览器侧边栏的 AI Agent。安装之后，你可以点击浏览器上的 Cebian 图标打开它：

![截图](./snapshot.png)

在完成 AI 提供商配置后，就可以开始对话了。

<Callout type="tip">
第一次使用时不需要一上来就配置 MCP 和 Skill。可以先试试着做一些一下页面感知能力，再逐步加深。
</Callout>

## 核心特点

- **本地优先**：对话、Prompt、Skill、附件都存在浏览器本地。除了发给模型厂商的必要请求，不向任何服务器上传数据。
- **直连你自己的模型**：填入任意厂商的 API Key 即可，不需要账号、不需要积分、也没有登录机制。
- **可扩展**：可以接入 MCP 工具*，也可以把常用的操作封装成 Skill。

> 注：MCP 只支持 http 协议，详情见 [MCP 章节](/zh/docs/guides/mcp/)。

## 能力示例

- 总结当前网页、长文、文档或 PR。
- 选中任意内容，让模型总结、翻译、改写。
- 抓取页面里的表格、列表、价格，转成 JSON 或 Markdown。
- 录制一遍操作，让 Agent 之后自己重放。
- 阅读 PDF 文档，回答相关问题。
- ……
