安装完成后，点击浏览器工具栏里的 Cebian 图标打开侧边栏。第一次使用时，如果还没配置模型，会看到「请先选择一个 AI 模型」。

你需要先给 Cebian 接一个 AI 提供商，然后选择一个模型，就可以开始对话了。

## 配置 AI 提供商

Cebian 本身不提供模型额度，也不需要注册 Cebian 账号。你可以使用自己已有的 API Key 或订阅。

目前主要有几种方式：

- 如果你有 GitHub Copilot 或 OpenAI Codex 订阅，可以在 OAuth 区域点击「登录」
- 如果你有 OpenAI、Anthropic、Google Gemini、DeepSeek、OpenRouter 等 API Key，可以在「通过 API Key」里填入并保存
- 如果你使用 Ollama、LM Studio 或自己搭的 OpenAI 兼容服务，可以在 OpenAI Compatible 里添加自定义提供商

普通使用的话，先配置一个自己最常用的提供商就行，MCP 和 Skill 都可以之后再看。

## 选择模型

配置完成后，回到新对话页面，在输入框左下角点击「选择模型」。

选择一个模型后，就可以在输入框里输入内容了。如果当前模型支持思考档，旁边还会出现「思考」选项；不知道选哪个的话，先保持默认就行。

## 发出第一条消息

你可以直接问当前页面相关的问题，比如：

- 总结一下当前页面
- 把我选中的内容翻译成中文
- 提取这个页面里的关键链接
- 看看我打开了哪些标签页，帮我整理一下

Cebian 会带上当前标签页的标题、URL、选中文本和标签页列表等上下文。如果需要读取页面正文，它会再按需调用工具去读。

## 可以先试试示例

配置好模型后，新对话页会出现几个示例卡片，比如「总结当前页面」「翻译当前页面」「提取页面要点」。

点击示例卡片后，它会把对应的内容填到输入框里，你可以直接发送，也可以先改成自己的说法再发送。

## 如果发不出去

- 提示「请先选择一个 AI 模型」：回到输入框左下角选择模型
- 提供商显示「未验证」：API Key 已保存，但连通性测试没通过，可以先试着发送一条消息看看
- 当前页面无法读取：Chrome 插件市场、浏览器设置页、部分特殊页面本来就不允许扩展读取内容，可以换一个普通网页再试
