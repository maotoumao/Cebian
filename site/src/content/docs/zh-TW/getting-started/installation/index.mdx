你可以選擇下面任意一種方式安裝 Cebian。普通使用的話，建議優先從外掛市場安裝；如果想第一時間用上最新版本，可以從 Github Release 下載。

## 通過外掛市場安裝

Chrome、Edge 以及大多數 Chromium 系瀏覽器都可以從外掛市場安裝：

- [Chrome 外掛市場](https://chromewebstore.google.com/detail/dnfmklmdbipiahdjiogaoemjaiodomdf)
- [Edge 外掛市場](https://microsoftedge.microsoft.com/addons/detail/kakfgcmbiacmclkjboijakkhlhioopcd)

開啟對應頁面後，點選安裝按鈕即可。安裝完成後，瀏覽器工具欄會出現 Cebian 圖示，點選它就可以開啟側邊欄。

> 商店版本需要等稽核，所以可能會比 Github Release 晚一點更新。

## 通過 Github Release 安裝

更新時也會在 Github 同步釋出，你可以去 Releases 中下載最新版本：

[Cebian Github Releases](https://github.com/maotoumao/Cebian/releases)

安裝步驟如下：

1. 下載 `cebian-{version}-chrome.zip`
2. 解壓到一個固定目錄
3. 開啟瀏覽器擴充套件管理頁面
4. 開啟「開發者模式」
5. 點選「載入已解壓的擴充套件程式」
6. 選擇剛才解壓出來的目錄

> Release 裡也會有 Firefox 包。如果你要在 Firefox 上手動安裝，可以下載對應的 `firefox.zip`，不過這裡先不展開 Firefox 的安裝流程。

## 安裝方式選擇

- 普通使用：外掛市場最省心
    - Edge 使用者：優先用 Edge 外掛市場
    - Chrome / Chromium 使用者：優先用 Chrome 外掛市場

- 想用最新版：Github Release 通常最快

## 安裝後

安裝完成後，點選瀏覽器工具欄裡的 Cebian 圖示開啟側邊欄。第一次使用時，需要先配置一個 AI 提供商，然後就可以開始對話。
