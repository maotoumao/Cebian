Cebian 是一個常駐瀏覽器側邊欄的 AI Agent。安裝之後，你可以點選瀏覽器上的 Cebian 圖示開啟它：

![截圖](./snapshot.png)

在完成 AI 提供商配置後，就可以開始對話了。

<Callout type="tip">
第一次使用時不需要一上來就配置 MCP 和 Skill。可以先試試著做一些一下頁面感知能力，再逐步加深。
</Callout>

## 核心特點

- **本地優先**：對話、Prompt、Skill、附件都存在瀏覽器本地。除了發給模型廠商的必要請求，不向任何伺服器上傳資料。
- **直連你自己的模型**：填入任意廠商的 API Key 即可，不需要賬號、不需要積分、也沒有登入機制。
- **可擴充套件**：可以接入 MCP 工具*，也可以把常用的操作封裝成 Skill。

> 注：MCP 只支援 http 協議，詳情見 [MCP 章節](/zh-TW/docs/guides/mcp/)。

## 能力示例

- 總結當前網頁、長文、文件或 PR。
- 選中任意內容，讓模型總結、翻譯、改寫。
- 抓取頁面裡的表格、列表、價格，轉成 JSON 或 Markdown。
- 錄製一遍操作，讓 Agent 之後自己重放。
- 閱讀 PDF 文件，回答相關問題。
